# ACE-B (Adaptive Cognitive Environment Benchmark)

RCS-1 (Rule Change Survival) is a benchmark designed to evaluate how agents adapt to hidden rule changes in non-stationary environments.

## Core Idea

The benchmark measures whether an agent can:

- learn a rule from interaction
- detect when the rule changes
- stop using outdated behavior
- adapt and stabilize

## Metrics

- PCM — Pre-Change Mastery  
- CDL — Change Detection Lag  
- PR — Perseveration Rate  
- AHL — Adaptation Half-Life
